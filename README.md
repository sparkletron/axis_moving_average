### Parameters

* BUS_WIDTH : DEFAULT : 1 : Width of the AXIS bus in/out.
* WEIGHT    : DEFAULT : 1 : Divisor for the average. Will only work with powers of 2 (1, 2, 4, 8, 16... etc)
